API Usage

This API aggregates several sources of legislator data. The current focus is on contact methods, but more data such as voting history will be added.

Currently restricted to national house/senate, but there are plans to implement state-level as well.

For bugs/questions/etc please contact speakunited.us@gmail.com

GET /contacts

Retrieve a list of contacts. Currently these are only house/senate legislators but may return other types of contacts in the future. If query params are provided, the list is filtered.

Query Params:

  • address: Address to query by. Returns only congressman that represent the state/district of that address
  • type: Current valid values are house and senate. Restricts results to congressman of the given type

Example:

$ curl "http://api.speakunited.us:8080/contacts?address=3333+Michelson+Drive+Irvine+CA&type=house"

GET /issues

Retrieve a list of current calls-to-actions along with their speaking/writing templates

$ curl "http://api.speakunited.us:8080/issues"

Contact Form Submittal

This api provides a forwarding service to https://congressforms.eff.org
